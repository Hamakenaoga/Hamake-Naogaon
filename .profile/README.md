# স্বাগতম! আমরা "হামাকে নওগাঁ" 🩸

নওগাঁ জেলার মানুষের জন্য একটি মানবিক ও অলাভজনক স্বেচ্ছাসেবী উদ্যোগ। আমরা বিশ্বাস করি, আমাদের ছোট একটি প্রচেষ্টা কারো জীবন বাঁচাতে বড় ভূমিকা রাখতে পারে।

### 🛠 আমাদের কার্যক্রম
- **জরুরি রক্তদান:** নওগাঁর ১১টি উপজেলার রক্তদাতাদের একটি শক্তিশালী নেটওয়ার্ক তৈরি করা।
- **তথ্য সেবা:** জরুরি প্রয়োজনে সঠিক তথ্যের মাধ্যমে সাধারণ মানুষের পাশে দাঁড়ানো।
- **স্বেচ্ছাসেবী নেটওয়ার্ক:** তরুণ প্রজন্মকে মানবিক কাজে উৎসাহিত করা।
